Material Design color controll attributes

	status bar 														-> colorPrimaryDark
	tool bar 	 														-> colorPrimary
	tool var title 												-> textColorPrimary
	application window/window background 	-> windowBackground
	Nevigation bar 												-> nevigationBarColor

	switch status off 										-> colorControllNormal default color -> colorAccent
	overflow menu 												-> textColorSecondary
	nevigationIcon 												-> textColorSecondary

	Tinting Widgets by default all widgets have colorAccent color 
		Buttons :
		ripple effect on button -> colorControlHighlight / default colorAccent
		
		CheckBox/Radio Buttons
		status when uncheck -> colorControlActivated
		status when check 	-> colorControllNormal
	
		Color palet :
		status bar : 700 tint
		tool bar 	 : 500 tint
		accent color : 200 tint
		
		accent color cant be same as primary color : 
		accent color not use for body text color
